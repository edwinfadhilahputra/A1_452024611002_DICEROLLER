🎲 Dice Roller App
Aplikasi Android sederhana untuk mensimulasikan lemparan dadu secara digital. Pengguna dapat menekan tombol Roll Dice untuk menghasilkan angka acak dari 1 hingga 6 dan menampilkan hasilnya dalam bentuk gambar dadu. Aplikasi ini dibangun menggunakan Kotlin dan Android Studio dengan antarmuka yang sederhana dan mudah digunakan.

📱 Fitur Utama
🎲 Melempar dadu secara acak (1–6)
🖼️ Menampilkan gambar sesuai hasil lemparan
🔄 Hasil berubah setiap kali tombol ditekan
📱 Antarmuka sederhana dan responsif
⚡ Ringan dan mudah digunakan
🛠️ Teknologi yang Digunakan
Kotlin
Android Studio
Jetpack Compose / XML Layout (sesuaikan dengan proyek)
Android SDK
